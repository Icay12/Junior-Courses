Java Lab3: miniCAD 
---------------------------
#requirement

编写一个简单的画图程序，能画以下四种图形：

直线：鼠标左键单击确定直线的第一个点，然后再单击确定直线的第二个点；

矩形：鼠标左键单击确定矩形的第一个角，然后再单击确定矩形的第二个角；

椭圆：鼠标左键单击确定包围椭圆的矩形的第一个角，然后再单击确定这个矩形的第二个角，为简单起见，只能画长短轴与坐标轴平行的椭圆；

文字：鼠标左键点击后，直接在图形窗口内输入文字，回车后成为图形。

其他要求有：

采用菜单选择要绘制的图形；

以数据方式记录图形，而不是记录图形的像素；

画下来的图形可以被选择，为简单起见，可以不处理重叠图形的选择问题；

被选中的图形可以通过del键被删除；

绘制的图形可以被保存为数据文件，文件可以被打开。

不要求能处理颜色、改变大小、字体等，也不要求复杂的交互界面，但是做这些可能有加分。
